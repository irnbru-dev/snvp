1. Логотип в меню должен быть ссылкой.
2. Иконка поиска — обсудили.
3. Иконка поиска появляется в мобильном меню — убрать.
4. Подвинуть выпадающее меню, для этого тут:
    @media (min-width: 992px) {
        .navbar .dropdown-menu {
            вместо margin-left: -15px; указать 15px
        }
    }
5. Из мобильного меню можно убрать поле поиска. На основном сайте нет поиска. Если говорят, что функционал не поменяется,
   то, значит, и поиска у них не будет.
6. Мобильное меню на планшетах имеет маленький отступ сверху. Даже на телефонах больше. Сделать отступ больше и
   одинаковым для моб и планшетов.
7. Для иконок в промо на мобильниках (.main-promo__item) делать padding: 0 и margin-top: 30-40 px.
8. Для иконок в футере на мобильниках и планшетах .footer__socials li
    {вместо margin-left: 30px; делать: margin-left: 15px; margin-right: 15px}
    чтобы они были по цетру.
9. В футере заголовки меню — тоже ссылки. Стиль оставить, как есть, ховер обрабатывать не надо.
10. На странице about-company.html фотки обернуты в .row по 3 штуки. Их надо все обернуть в один общий .row, а другие
    .row убрать.
